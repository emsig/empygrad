# empygrad

> **Under active development.** APIs may change without notice.

Open-source full 3D electromagnetic analytical gradients for 1D VTI media, built on top of [empymod](https://empymod.emsig.xyz).

**empygrad** computes analytical Jacobians for 1D VTI (Vertical Transverse Isotropic) electromagnetic media. Currently supported gradients include resistivity (`res`) and anisotropy (`aniso`). Future releases will extend coverage to a full set of gradients, including geometry parameters (layer depths, thicknesses), source/receiver positions, and orientation parameters such as tilt angle.
